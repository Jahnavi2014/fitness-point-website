
# Fitness Point Website

Fitness Point is a modern fitness website developed using HTML, CSS, and JavaScript. It provides users with various workout categories and essential fitness-related features in a simple and responsive interface.

## 🌟 Features
- Multiple workout categories:
  - Cardio
  - Boxing
  - Yoga
  - Pilates
  - HIIT (High-Intensity Interval Training)
- Gallery page to showcase fitness activities
- Membership signup page
- Fitness exploration section
- Fully responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
